un post con diferencias de código

27 2024

2024   ·   formato   código   ·   sample-posts

Puedes mostrar el código de diferencias utilizando la sintaxis markdown normal:

```diff
diff --git a/sample.js b/sample.js
index 0000001..0ddf2ba
--- a/sample.js
+++ b/sample.js
@@ -1 +1 @@
-console.log("¡Hola Mundo!")
+console.log("¡Hola desde Diff2Html!")
```

Lo cual genera:

diff --git a/sample.js b/sample.js
index 0000001..0ddf2ba
--- a/sample.js
+++ b/sample.js
@@ -1 +1 @@
-console.log("¡Hola Mundo!")
+console.log("¡Hola desde Diff2Html!")

Pero esto es difícil de leer, especialmente si tienes una diferencia grande. Puedes usar diff2html para mostrar una versión más legible de la diferencia. Para esto, solo usa diff2html en lugar de diff para el lenguaje del bloque de código:

```diff2html
diff --git a/sample.js b/sample.js
index 0000001..0ddf2ba
--- a/sample.js
+++ b/sample.js
@@ -1 +1 @@
-console.log("¡Hola Mundo!")
+console.log("¡Hola desde Diff2Html!")
```
